# Restaurant Management System

Cloud-based restaurant operations platform built with React + Supabase.

## What is included

- POS-style order management (dine-in, takeaway, delivery)
- Kitchen ticket tracking
- Table and reservation management
- Menu and inventory control
- Guest profiles and staff management
- Reporting dashboard
- English + Arabic support

## Project structure

- `frontend/` - React application
- `database/schema.sql` - Supabase/PostgreSQL schema
- `DEPLOYMENT_GUIDE.md` - setup and deployment instructions
